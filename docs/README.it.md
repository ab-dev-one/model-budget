# Documentazione di ModelBudget

Questa cartella contiene la documentazione del progetto ModelBudget in italiano.

## Panoramica

ModelBudget è un'applicazione web open source per stimare e confrontare i costi dei modelli AI prima di distribuire funzionalità di produzione. Il progetto è progettato per rimanere completamente lato client, rispettare la privacy e essere gratuito.

## Obiettivi principali

- Stimare il costo di prompt e output
- Confrontare modelli tra loro
- Simulare crescita e impatto sul budget
- Suggerire opportunità di ottimizzazione
- Condividere scenari senza backend

## Architettura

- Solo frontend
- Nessun accesso autenticato
- Nessun tracciamento
- Nessun backend richiesto
- Dati di pricing gestiti in file JSON o strutture tipizzate
