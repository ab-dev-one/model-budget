# Contribuire

Grazie per contribuire a ModelBudget.

## Setup di sviluppo

1. Installa Node.js 20+
2. Esegui `npm install`
3. Esegui `npm run dev`

## Controlli di qualità

- `npm run lint`
- `npm run test`
- `npm run build`

## Aspettative per la pull request

- Mantieni i cambiamenti focalizzati
- Aggiungi o aggiorna i test quando serve
- Documenta i cambiamenti visibili all'utente
