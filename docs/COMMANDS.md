# Commands

## Install dependencies

```bash
npm install
```

## Start local development server

```bash
npm run dev
```

## Run tests

```bash
npm run test
```

## Run tests in watch mode

```bash
npm run test:watch
```

## Lint the project

```bash
npm run lint
```

## Build for production

```bash
npm run build
```

## Preview the production build

```bash
npm run preview
```

## Format the code

```bash
npm run format
```
