# ModelBudget Documentation

This directory contains the project documentation for ModelBudget in English.

## Overview

ModelBudget is an open-source web application for estimating and comparing AI model costs before shipping production features. The project is designed to remain fully client-side, privacy-friendly and free to use.

## Main goals

- Estimate prompt and output costs
- Compare models side by side
- Simulate growth and budget impact
- Suggest optimization opportunities
- Share scenarios without a backend

## Architecture

- Frontend only
- No authentication
- No tracking
- No backend required
- Pricing data stored as JSON or typed data files
