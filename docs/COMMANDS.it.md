# Comandi

## Installare le dipendenze

```bash
npm install
```

## Avviare il server di sviluppo locale

```bash
npm run dev
```

## Eseguire i test

```bash
npm run test
```

## Eseguire i test in modalità watch

```bash
npm run test:watch
```

## Eseguire il lint del progetto

```bash
npm run lint
```

## Eseguire la build di produzione

```bash
npm run build
```

## Visualizzare la build di produzione

```bash
npm run preview
```

## Formattare il codice

```bash
npm run format
```
