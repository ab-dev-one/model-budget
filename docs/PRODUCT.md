# Product Specification

## Vision

ModelBudget helps developers estimate, compare and optimize AI model costs before writing production code.

## Mission

Provide a free, open-source and privacy-friendly planning tool for evaluating AI cost trade-offs.

## Product philosophy

- No login
- No backend
- No tracking
- No ads
- Runs entirely in the browser
- Open source first

## Target users

- Indie developers
- Freelancers
- Startups
- Software houses
- AI engineers
- Students

## Goals

- Estimate AI costs
- Compare models
- Simulate growth
- Suggest optimizations
- Share scenarios

## Non-goals

- Calling LLM APIs
- User accounts
- Billing
- Provider dashboards

## Technical stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Vitest
- GitHub Actions
- GitHub Pages

## Architecture principles

- Frontend only
- Business logic separated from UI
- No backend dependency
- Privacy by design
